วิธีการเล่น

เริ่มเล่นเบสง่ายๆ ใน 3 ขั้นตอน

เบสเป็นเครื่องดนตรีที่เปรียบเสมือนผู้ปิดทองหลังพระ ทำหน้าที่ควบคุมจังหวะของวง ทำให้ดนตรีดูแน่นขึ้น  ถือว่าเป็นเครื่องดนตรีอีกชิ้นที่สำคัญมากๆ ในวงดนตรี

หลายๆคนอาจคิดว่าเบสนั้นเล่นง่ายกว่ากีตาร์เพราะมีจำนวนสายที่น้อยกว่า แต่ความจริงแล้วเบสเองก็มีเทคนิคยากๆที่ไม่ต่างจากกีตาร์เลย เรียกได้ว่าถ้าจะเล่นให้เนียนก็ยากพอตัวเหมือนกัน  แต่ในวันนี้ Music Arms จะพาไปดู 3 ขั้นตอนง่ายๆ ที่เราจะสามารถเริ่มเล่นเบสได้อย่างมีประสิทธิภาพ และนำไปใช้งานได้จริงแน่นอน  จะมีอะไรบ้างไปเริ่มกันได้เลย

  1. เรียนรู้โน๊ต Root ของคอร์ด

ในการเล่นเบส เราจะเล่นเป็นโน๊ตมากกว่าเล่นเป็นคอร์ด โดยโน๊ตส่วนใหญ่ที่เล่นจะเป็นโน๊ต Root ดังนั้นเราต้องรู้ว่า Root ของคอร์ดอยู่ตรงไหนคือโน๊ตอะไร ซึ่งส่วนมากจะเป็นโน๊ตที่ชื่อเดียวกับคอร์ดนั้นๆ เช่น คอร์ด C โน๊ต Root ก็คือ C คอร์ด Am โน๊ต Root ก็คือ A นั่นเองแต่ในกรณีที่เป็นคอร์ด on เช่น D/F# , C/D , C/B คอร์ดเหล่านี้ให้ดูที่ตัวหลัง / โน๊ตตัวนั้นจะเป็นโน๊ต Bass  ของคอร์ดนั่นเองโน๊ต Root ส่วนใหญ่จะเป็นตัวเดียวกับชื่อคอร์ดเช่น คอร์ด C โน๊ต root ก็จะเป็น Cแต่ในกรณีที่เป็นคอร์ด / ให้ดูโน๊ตตัวที่อยู่หลัง / เป็นหลักเพราะจะเป็นโน๊ต Bass ของคอร์ด

 

  1. ฟังเสียงกระเดื่องให้ออก

ข้อนี้สำคัญมากเพราะเบสจะเล่นคู่กับกระเดื่องเป็นหลักถ้าหากเรา แยกเสียงกระเดื่องไม่ออกจะทำให้เราเล่นผิดจังหวะได้

โดยเริ่มแรกเราอาจฟังเพลงที่จะแกะให้มากขึ้น แล้วจับเสียงกระเดื่องให้ได้ จากนั้นให้เริ่มเล่นตามอาจจะไม่ต้องเล่นเป็นคอร์ดในทันทีเลยก็ได้แต่ให้จำว่าในแต่ล่ะห้องนั้นกระเดื่องเยียบกี่ครั้ง จากนั้นก็ลองใส่คอร์ดลงไปตามจังหวะ

ฟังเสียงกระเดื่องของเพลงที่จะแกะจนจำจังหวะในการเยียบของกระเดื่องได้ แล้วลองเล่นตาม

 

  1. คุมน้ำหนักมือให้ดี

น้ำหนักมือในการดีดจะส่งผลต่ออารมณ์ของเพลงเป็นอย่างมาก รวมถึงความแข็งแรงของเพลงด้วยเพราะถ้าหากเราเล่นน้ำหนักไม่สม่ำเสมอจะทำให้เกิดอาการเสียงเบสหายไปในบางช่วงหรือเสียงเบสดังขึ้นมาก เป็นบางจังหวะ เพราะฉนั้นแล้วมือเบสทุกคนควรคำนึงถึงน้ำหนักในการดีดให้มาก วิธีฝึกง่ายๆ อาจเริ่มจากการดีดช้าๆ ให้น้ำหนักเท่ากันจากนั้นก็ค่อยๆ เพิ่มความเร็วขึ้น โดยยังคงน้ำหนักเดิมไว้ ทำซ้าวนไปเรื่อยๆ จนกล้ามเนื้อของเราจำความรู้สึกได้ ก็จะสามารถคุมน้ำหนักของมือได้แล้ว  หลักสำคัญที่ห้ามลืมเลยคือเปิดเมโทรนอมระหว่างฝึกไปด้วย

ต้องคุมน้ำหนักของการดีดในแต่ล่ะครั้งให้เท่ากัน เพราะถ้าดีดเบาบ้างดังบ้าง อารมณ์ของเพลงจะเปลี่ยนไปและเสียงเบสจะฟังออกมาไม่ชัด

เป็นไงกันบ้างครับสำหรับ 3 วิธีง่ายๆที่จะทำให้เพื่อนๆ เริ่มเล่นเบสได้อย่างมีประสิทธิภาพ สามารถนำไปฝึกกันได้เลยนะครับรับรองได้เลยว่าเล่นเบสได้แน่นอน หากเพื่อนคนไหนมีคำถามสามารถคอมเม้นต์ไว้ได้เลยครับ สุดท้ายนี้ขอให้สนุกกับการเล่นดนตรีครับ และหากผิดพลาดประการใดต้องขออภัยมา ณ ที่นี้ด้วยครับ
